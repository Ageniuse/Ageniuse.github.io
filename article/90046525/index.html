    • 首先找到菜单栏中的Window 下的Preference

    • 然后找到Server 下的 Runtime Environment

    • 然后点击左侧的Add,因为小编用的是Tomcat 9.0 的，你的就要看自己的了，然后Next

    • 然后找到自己存放Tomcat 的路径，然后Finish ，最后Apply and Close 即可

    • 其实上一步就可以完成Tomcat 的配置了，接下来的步骤是验证项目在Tomcat 是否部署成功，点击File ,然后New，创建个动态网站Dynamic Web Project

    • 这一步中项目名命个名即可，其他设置默认即可，然后Next

    • 这一步直接Next

    • 将勾选选上，将Content directory 修改为WebRoot,然后Finish 即可

    • 然后右击WebRoot ,选择New ,然后选额JSP File

    • 然后修改一下File name ,然后给创建的jsp<body>标签里添加行代码<h1>HelloWorld</h1>  记住该jsp 文件必须是在WebRoot根目录下，要不然会出现404 错误

    • 然后点击那个绿色的启动按钮，在弹出的页面直接点击Finish 即可
